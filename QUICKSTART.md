### 3단계: 워크플로우 실행 (5분)

#### 🔧 API 설정 섹션 (상단)
1. **AI 서비스**: "Google Gemini" 선택
2. **API 키**: 발급받은 키 입력
3. **🧪 API 연결 테스트** 버튼 클릭 → 성공 확인

#### 📰 1. 뉴스 수집
- 키워드 입력 (예: `삼성,쿠팡,부동산`)
- "🔍 뉴스 수집 시작" 클릭
- 목록에서 기사 **클릭하여 선택**
- ⭐ **중요**: 다른 기사로 변경하려면 "🔄 새로고침" 클릭!

#### ✍️ 2. AI 대본 생성
- "📝 대본 생성" 클릭
- 30초 대기
- **생성된 대본이 화면에 표시됨** ✅
- "📖 전체 대본 보기" 클릭하면 전체 내용 확인 가능

#### 🎨 3. 썸네일 제목
- "🎯 썸네일 제목 10개 생성" 클릭
- 10개 제목 중 하나 클릭하여 선택
- **대본 내용과 연관된 제목 생성됨** ✅

#### 🎤 4. TTS 음성 생성
- "🔊 음성 생성" 클릭
- **생성된 파일 정보 표시됨** ✅
  - 파일명, 길이, 크기 확인 가능

#### 🎬 5. 영상 편집
- "✂️ 영상 편집 시작" 클릭
- **편집된 영상 정보 표시됨** ✅
  - 해상도, 길이, 크기 확인 가능

#### 📤 6. 유튜브 업로드
- **"👀 최종 확인" 버튼 먼저 클릭!** ✅
  - 모든 내용 검토 가능
  - 뉴스, 대본, 썸네일, 음성, 영상 모두 확인
- "🚀 유튜브 업로드" 클릭

## 🐛 문제 해결

### Q1: Gemini API 연결이 안 돼요

**증상**: "❌ API 연결 실패" 메시지

**해결 방법**:
```bash
# 1. 터미널에서 테스트
python test_api.py

# 2. 오류 메시지 확인
# - 403 오류: API 키가 잘못됨 → 새 키 발급
# - 400 오류: 키 형식 오류 → 공백 제거 후 재입력
```

**Gemini API 키 재발급**:
1. https://makersuite.google.com/app/apikey
2. 기존 키 삭제
3. "Create API key in new project" 클릭
4. 새 키 복사

---

### Q2: 대본이 안 보여요

**원인**: API 응답 파싱 오류

**해결 방법**:
1. `dashboard_v2.html` 사용 (구버전 아님!)
2. 브라우저 콘솔 확인 (F12 → Console 탭)
3. 오류 메시지 복사하여 확인

**Mock 모드로 테스트**:
- API 키 입력하지 않고 바로 사용
- 정상 작동하면 API 키 문제

---

### Q3: 뉴스가 변경되지 않아요

**해결 방법**:
1. **"🔄 새로고침" 버튼 클릭** ← v2에서 추가됨!
2. 키워드 변경 후 "🔍 뉴스 수집 시작" 재클릭
3. 수집 시간 확인 (최신 시간이어야 함)

---

### Q4: 썸네일 제목이 뉴스와 관련없어요

**원인**: 구버전 사용 또는 API 미연동

**해결 방법**:
1. **`dashboard_v2.html` 사용 확인**
2. Gemini API 키 입력 확인
3. "🧪 API 연결 테스트"로 연결 확인
